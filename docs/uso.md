## Casos de uso
- Crear registros nuevos
- Consultar información existente
- Editar o eliminar datos
- Navegar entre módulos

## Usuarios tipo
- **Estudiante:** utiliza la aplicación para gestionar y visualizar información de práctica.
- **Profesor:** supervisa, revisa o evalúa el uso del sistema por parte de los estudiantes.

## Explicación técnica
El sistema se divide en tres capas principales:

- **Frontend:** interfaz desarrollada en HTML, CSS y JavaScript.
- **Backend:** gestión lógica y procesamiento de datos.
- **Base de datos:** almacenamiento de registros y consultas.

La comunicación entre módulos se realiza mediante peticiones HTTP.

## Tecnologías
- HTML5  
- CSS3  
- JavaScript  
- Node.js
